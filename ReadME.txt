В папке done выполенное задание робота
В папке config вводим данные для почты smtp
РАБОТА РОБОТА :
Робот запускается из процесса "\Новый Проект\main."
читает "\Новый Проект\Список банков".
открывает браузер.
переходит по ссылке,имитируя пользователя, забирает данные с сайта.
Используя регулярные выражения обрабатывает данные. 
Сохраняет в "\Новый Проект\done\список банков по запросу.xlsx" полученые данные.
переходит в процесс "\Новый Проект\mailSender."
читает "\Config\Config.xlsx"
ожидает от пользователя ввод почты куда нужно направить.
Отправляет данные на почту с вложением "список банков по запросу.xlsx".

Перед началом активации робота необходимо проверить пути к файлам excel для активностей excel 
и заменить на текущие для конкретного компьютера.Для переход из"main" в "mailSender"тоже стоит 
проверить путь у активности"запустить процесс".
Относительные пути могут не сработать корректно.
